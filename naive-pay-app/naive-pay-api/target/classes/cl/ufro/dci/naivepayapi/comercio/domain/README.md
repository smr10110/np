## Domain Packgage
This package defines and models the main entities of commerces.

## Main Classes

- **Commerce**: This class represents the commerces in the system.
- **CommerceCategory**:This class represents de categories of commerces in the system.
- **CommerceInfo**:Embeddable class, contains basic information of the commerce.
- **CommerceValidation**:Represents and income commerce which requests validation. 
- **ValidationStatus**:Represents a commerce's validation request.
- **VerificationStatus**:Represents a commerce's verification status.
