## Dto Packgage
This package contains objects for transferring data between the frontend and banckend.
## Main Classes

- **CommerceApproval**: This class approves commerces
- **CommerceCreation**:This class creates commerces.
- **CommerceResponse**:This class is used to respond to the frontend.
- **CommerceUpdate**:This class is used to update the commerces in the system.
- **CommerceValidationRequest**:This class send a validation request from a commerce.
